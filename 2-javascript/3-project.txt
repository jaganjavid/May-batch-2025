
===========================
Console-Based JS Projects
===========================

These are beginner-friendly projects designed to help reinforce JavaScript fundamentals
(variables, arrays, objects, conditionals, loops, functions, etc.) using only the console (no DOM).

----------------------------------------------------
1. Student Grading System (Console App)
----------------------------------------------------
Description:
- Create an array of student objects (name, marks).
- Write a function to calculate grades based on marks.
- Print each studentâ€™s name, marks, and grade (A/B/C/Fail).

Bonus:
- Use switch-case to show different messages for each grade.

----------------------------------------------------
2. Mini Bank Account Simulator
----------------------------------------------------
Description:
- Use an object to store 'balance', 'name', and 'transactions'.
- Create functions: deposit(), withdraw(), checkBalance().
- Store all transactions in an array.
- Display final balance and full transaction history in console.

----------------------------------------------------
3. Todo Manager (Console Version)
----------------------------------------------------
Description:
- Add a todo using addTodo("Task Name")
- Delete a todo by index or name.
- List all todos.
- Mark todo as done using a status flag (true/false).
- Use array of objects to manage todos.

Goal:
- Practice JavaScript logic without relying on the browser UI.
